Chinese Character Recognition

Hello! This is where I hope to host my machine learning model in the future. Project hosted at: https://github.com/tyler-pruitt/Chinese-Character-Recognition. The idea is to create a space (i.e. a 64x64 pixel box) in which anyone can draw one of the Chinese characters that the model has been trained on and it will make a prediction! I also plan to showcase the probabilities with a numeric and graphical output of the model's prediction to show some cool statistics! Please keep coming back to see updates and progress!

To use:
1) Draw the character in the black box below. Available characters are 零，一，二，三，四，五，六， 七，八，九，十，百，千，万，and 亿.
2) Download your drawing either as an PNG or JPG image
3) Upload this image to have the model predict what character it is
