Chinese Character Recognition

Hello! This is where I hope to host my machine learning model in the future.
The idea is to create a space (i.e. a 64x64 pixel box) in which anyone can draw one of the Chinese characters that the model has been trained on and it will make a prediction!
I also plan to showcase the probabilities with a numeric and graphical output of the model's prediction to show some cool statistics!
Please keep coming back to see updates and progress! Thanks!
